# My_AI_Assistant
A modern AI assistant built using Streamlit with:

✅ Chat interface  
✅ Voice input  
✅ AI responses  
✅ Chat history  
✅ Text-to-speech  
✅ Modern UI  

---

##  Features

- ChatGPT-like interface
- Voice input support
- AI text response
- Chat history saving
- Stop response button
- Sidebar history

---

##  Tech Stack

- Python
- Streamlit
- Groq API
- SpeechRecognition
- HTML/CSS
- JavaScript
